This add-on enables the login model for the WPSPX plugin

== Description ==

This plugin adds a login box to your website, allowing your customers to login to thier spektrix customer account before visiting the checkout. Useful if you offer membership based discounts or promotions. Requires WPSPX plugin.

== Features ==

- Add the login link to the end of any WordPress nav menu defined in WPSPX > Login
- A PHP function `wpspx_login_link()` if you want to add it to your template
-

== Installation ==

- Upload wpspx-login to the /wp-content/plugins/ directory and activate
- Activate the plugin through the 'Plugins' menu in WordPress
- Update your plugin settings choosing a WordPress nav menu location
- or Add the php function to your template where you want the link to appear.
