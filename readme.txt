== Description ==

This plugin allows you to quickly integrate your Spektrix API data into your WordPress website.

== Features ==

- Shortcode `[wpspx_login_link]` displays a login link
- php function `wpspx_login_link()` if you want to add it to your template

== Installation ==

- Upload wpspx-login to the /wp-content/plugins/ directory and activate
- Activate the plugin through the 'Plugins' menu in WordPress
- Update your plugin settings choosing colours &amp; menu location
- or Add shortcode or php function to your template where you want the link to appear.
